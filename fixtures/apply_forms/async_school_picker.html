Education

Reproduces a Greenhouse-style async School picker: the OPEN menu (empty query) shows a default slice of schools ALPHABETICALLY — never the applicant's — so a batch pick over it declines. Typing a query fetches matching schools; the full résumé value ("The Pennsylvania State University") is a prefix miss (returns nothing, like the real index), but the article-stripped query returns the two Penn State campuses. The BRANCH campus is listed first, so a first-fuzzy-match substring fallback would commit the wrong campus — the Claude pick must prefer the main one (decision 080).
